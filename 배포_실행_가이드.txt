========================================
ELBIX AIDD 배포 실행 가이드
========================================

현재 환경에서는 Docker 데몬을 자동으로 시작할 수 없습니다.
실제 서버에서 다음 명령어를 실행하세요.

----------------------------------------
1단계: 서버에 SSH 접속
----------------------------------------
ssh user@your-server-ip

----------------------------------------
2단계: 프로젝트 디렉토리로 이동
----------------------------------------
cd /root/elbix_aidd

----------------------------------------
3단계: Docker 서비스 시작
----------------------------------------
sudo systemctl start docker
# 또는
sudo dockerd &

# 확인
docker ps

----------------------------------------
4단계: 배포 실행
----------------------------------------
# 방법 1: 자동 스크립트
./RUN_DEPLOYMENT.sh

# 방법 2: 수동 실행
docker compose up -d --build

----------------------------------------
5단계: 상태 확인
----------------------------------------
docker compose ps
docker compose logs -f

----------------------------------------
6단계: SSL 인증서 설정 (HTTPS - 무료)
----------------------------------------
sudo apt-get update
sudo apt-get install -y certbot python3-certbot-nginx
sudo certbot --nginx -d stlogic.aidd.co.kr

----------------------------------------
7단계: DNS 설정 확인
----------------------------------------
도메인 stlogic.aidd.co.kr이 서버 IP로 설정되어 있는지 확인:
nslookup stlogic.aidd.co.kr

========================================
완료!
========================================
접속: https://stlogic.aidd.co.kr
